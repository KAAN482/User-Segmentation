# Antigravity Transcript - User Segmentation Project

## Project Overview
This project implements a user segmentation engine using Flask and in-memory SQLite. It evaluates users against dynamic SQL-like rules.

## Timeline of Actions

1.  **Initialization**:
    -   Setup git, venv, and structure.
    -   Connected to GitHub repo.

2.  **Core Implementation**:
    -   Created Flask app (`app.py` - formerly `main.py`).
    -   Implemented `validator.py` for input checking.
    -   Implemented `evaluator.py` for segmentation logic (SQLite + `_now()` replacement).

3.  **UI & Testing**:
    -   Created `test.html` with a split-pane JSON editor for manual testing.
    -   Created `examples/` directory with JSON test cases.
    -   Documented `manual_tests.md` for curl/PowerShell/CMD testing.

4.  **Deployment**:

5.  **Documentation & Polish**:
    -   Added English comments to all Python files.
    -   Cleaned up unused files and imports.
    -   Updated `README.md` with comprehensive instructions.

## File Structure
- `app.py`: Main entry point.
- `evaluator.py`: Rule engine.
- `validator.py`: logic validation.
- `test.html`: Testing UI.
- `examples/`: Sample JSON payloads.
- `Dockerfile`: Container config.
- `requirements.txt`: Dependencies.
